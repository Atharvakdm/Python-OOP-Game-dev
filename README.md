# Python Learning Projects & Games

A collection of Python and Lua projects developed during high school, showcasing object-oriented programming (OOP) principles, game development, and data analysis techniques.

## 📁 Project Overview

This repository contains various interactive programs and learning projects, including:

### 🎮 Games & Interactive Programs
- **Atharva's Health Spot!** - Interactive menu-driven program with mini-games (Guess the number, Rock-Paper-Scissors)
- **The Space Quest (in progress)** - Text-based adventure game with narrative choices
- **Guess the number!** - Simple number guessing game
- **Rock-Paper-Scissors-with-Text** - Classic game implementation
- **Quirky Name Generator** - Fun name generation utility
- **Quiz.py** - Interactive quiz program

### 📊 Data & Analysis Projects
- **f1_predictor.py** - F1 Racing data analysis tool that predicts race winners based on historical team performance using pandas groupby operations and statistical analysis
- **read_csv.py** - CSV file parser with data aggregation and averaging functionality
- **cv-image-detect.py** - Computer vision image detection script

### 📚 Educational Programs
- **oop-book.py** - Object-oriented programming concepts demonstration
- **Task-Spot-code.py** - Task management utility
- **Galactic_adventure.py** - Interactive adventure game
- **pytorch.py** - PyTorch learning experiments

### 🎮 Game Development
- **Roblox_interface.lua** - Lua scripting for Roblox game development

## 📝 Usage

Each Python script can be run independently:

```bash
python filename.py
```

For data analysis projects, ensure required CSV files are in the same directory:

```bash
python f1_predictor.py
```
